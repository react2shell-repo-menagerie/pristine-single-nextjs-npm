# single-nextjs-npm - PRISTINE

This project contains pristine versions of React/Next.js packages without any known vulnerabilities.

## Project Type
Simple Next.js app with npm package manager

## Variant: pristine

### Dependencies
```json
{
  "nextVersion": "^15.5.7",
  "reactVersion": "^18.3.1",
  "reactDomVersion": "^18.3.1"
}
```

## Testing

To check for vulnerabilities:
```bash
npm test
```

To apply fixes:
```bash
npm run fix
```

## Setup

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) with your browser.
